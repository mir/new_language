# B1 - Πιστοποίηση Επάρκειας της Ελληνομάθειας

## Παραγωγή Προφορικού Λόγου - Δεύτερη Σειρά Δειγμάτων (02)

**25 Μονάδες | Διάρκεια εξέτασης: 12 λεπτά**

---

## Πρώτο Μέρος

- Πώς σας λένε;
- Πού κατοικείτε;
- Πηγαίνετε σχολείο; Σε ποια τάξη; (για μαθητές)
- Εργάζεστε; (για ενήλικες)
- Έχετε ελεύθερο χρόνο; Πώς περνάτε τον ελεύθερο χρόνο σας;
- Έχετε επισκεφθεί την Ελλάδα; Έχετε φίλους και συγγενείς εκεί;
- Μιλάτε άλλες ξένες γλώσσες εκτός από τα ελληνικά;
- Ποιο είναι το αγαπημένο σας φαγητό;

---

## Δεύτερο Μέρος

1. Ποια είναι η γνώμη σας για την τηλεόραση; Είναι καλό ή κακό να βλέπουν τα παιδιά τηλεόραση; Γιατί;
2. Πού προτιμάτε να ψωνίζετε; Σε εμπορικά κέντρα ή σε μικρά μαγαζιά; Γιατί; Ποια είναι τα υπέρ και τα κατά του ενός και ποια του άλλου;
3. Τι μπορούμε να κάνουμε για να έχουμε καλές σχέσεις με τους άλλους;
4. Στην Ελλάδα, αλλά και σε άλλες χώρες υπάρχουν πολλά σκουπίδια στους δρόμους. Πώς μπορούμε να τα μειώσουμε;
5. Ανακυκλώνετε υλικά; Δικαιολογήστε την απάντησή σας.
6. Σας αρέσει να ακούτε μουσική; Νομίζετε ότι η μουσική επηρεάζει τη διάθεση των ανθρώπων; Αν ναι, με ποιον τρόπο; Γιατί;
7. Ποια θεωρείτε ότι είναι τα θετικά και ποια τα αρνητικά στοιχεία του διαδικτύου;

---

## Εικόνες (Οδηγίες)

Δίνετε σε κάθε υποψήφιο/υποψήφια μια διαφορετική εικόνα. Ο υποψήφιος/Η υποψήφια βλέπει την εικόνα για ένα (1) λεπτό και στη συνέχεια του ζητάτε να την περιγράψει.

Ο εξεταστής/Η εξετάστρια μπορεί να θέσει ερωτήσεις προκειμένου να βοηθήσει τους υποψηφίους σε περίπτωση που αντιληφθεί ότι δεν μπορούν να προχωρήσουν στην περιγραφή της εικόνας.

Δεν δίνονται όλες οι φωτογραφίες σε κάθε υποψήφιο. Χρησιμοποιούνται δύο (2) αντίγραφα της κάθε φωτογραφίας. Ένα για τον εξεταστή με τις ερωτήσεις και ένα για τον εξεταζόμενο χωρίς τις ερωτήσεις.

### Εικόνα 1

*[Φωτογραφία: Άνθρωποι σε γυμναστήριο κάνουν ποδήλατο]*

**Ερωτήσεις:**
- Τι βλέπετε στην εικόνα;
- Που βρίσκονται οι άνθρωποι; Πως νοιώθουν;
- Εσείς πηγαίνετε σε γυμναστήριο; Ασχολείστε με κάποιο άθλημα;
- Τι μας προσφέρει;
- Τι άλλο κάνετε για να είστε υγιής;

### Εικόνα 2

*[Φωτογραφία: Οικογένεια (γονείς και παιδί) βλέπουν τηλεόραση στον καναπέ]*

**Ερωτήσεις:**
- Τι βλέπετε στην εικόνα;
- Εσείς βλέπετε τηλεόραση;
- Ποιες εκπομπές σας αρέσουν;
- Βλέπετε τηλεόραση με τους γονείς σας;
- Προτιμάτε να δείτε τηλεόραση ή να πάτε βόλτα με τους φίλους σας; Γιατί;

## Τρίτο Μέρος - Παιχνίδι Ρόλων

### 1. Στο κολυμβητήριο

**Α Ρόλος:**
Πηγαίνετε στο κολυμβητήριο του δήμου και ρωτάτε για τα μαθήματα κολύμβησης που γίνονται εκεί (ώρες μαθημάτων, προγράμματα προπονήσεων κτλ.). Ρωτάτε, επίσης, τι πρέπει να αγοράσετε, για να αρχίσετε να πηγαίνετε στην πισίνα.

**Β Ρόλος:**
Εργάζεστε σε ένα δημοτικό κολυμβητήριο ως γραμματέας και ένας κύριος/μια κυρία ζητάει να μάθει διάφορες πληροφορίες για τα μαθήματα κολύμβησης που γίνονται εκεί (ώρες μαθημάτων, προγράμματα προπονήσεων, κτλ.) και τι πρέπει να αγοράσει για αυτά. Απαντάτε σε όλες τις ερωτήσεις του/της και επιπλέον του/της λέτε ότι πρέπει να κάνει και ιατρικές εξετάσεις.

### 2. Διοργάνωση σχολικής γιορτής στο τέλος της χρονιάς

**Α Ρόλος:**
Εργάζεστε ως δάσκαλος σε ένα δημοτικό σχολείο και συζητάτε με έναν/μια συνάδελφό σας για το πώς θα διοργανώσετε μια γιορτή για το τέλος της σχολικής χρονιάς: για τους μαθητές που θα χορέψουν, τους μαθητές που θα τραγουδήσουν, για τον στολισμό της αίθουσας εκδηλώσεων και για όλα τα άλλα που πρέπει να ετοιμάσετε.

**Β Ρόλος:**
Εργάζεστε ως δάσκαλος σε ένα δημοτικό σχολείο και συζητάτε με έναν/μια συνάδελφό σας για το πώς θα διοργανώσετε μια γιορτή για το τέλος της σχολικής χρονιάς: για τους μαθητές που θα χορέψουν, τους μαθητές που θα τραγουδήσουν, για τον στολισμό της αίθουσας εκδηλώσεων και για όλα τα άλλα που πρέπει να ετοιμάσετε.

### 3. Οικογενειακό ή σπορ αυτοκίνητο;

**Α Ρόλος:**
Χρειάζεστε ένα αυτοκίνητο και συζητάτε με έναν φίλο/μια φίλη σας για το είδος του αυτοκινήτου που θα σας ταίριαζε καλύτερα. Εσείς θέλετε να αγοράσετε ένα μεγάλο οικογενειακό αυτοκίνητο, για να χωράνε πολλοί άνθρωποι μέσα και να μπορείτε να μεταφέρετε περισσότερα πράγματα. Ο φίλος/Η φίλη σας, όμως, σας λέει ότι θα σας ταίριαζε ένα σπορ αυτοκίνητο και σας εξηγεί γιατί. Εσείς επιμένετε ότι χρειάζεστε ένα οικογενειακό αυτοκίνητο.

**Β Ρόλος:**
Ένας φίλος/Μια φίλη σας χρειάζεται ένα αυτοκίνητο και συζητάτε μαζί του/της για το είδος του αυτοκινήτου που θα του/της ταίριαζε περισσότερο. Ο φίλος/Η φίλη σας θέλει να αγοράσει ένα μεγάλο οικογενειακό αυτοκίνητο και σας εξηγεί γιατί. Εσείς, όμως, του/της εξηγείτε ότι αφού δεν έχει ακόμη οικογένεια, καλύτερα θα ήταν να έπαιρνε ένα σπορ αυτοκίνητο, για να μπορεί να μετακινείται γρήγορα και να κάνει τις δουλειές του/της. Εκείνος/Εκείνη, όμως, επιμένει ότι χρειάζεται ένα οικογενειακό αυτοκίνητο. Προσπαθείτε να τον/την πείσετε ότι δεν έχει δίκιο.

### 4. Γυμναστική στο πάρκο ή στο γυμναστήριο;

**Α Ρόλος:**
Τηλεφωνείτε στην υπηρεσία αθλητισμού του δήμου και ζητάτε πληροφορίες για χώρους, όπου μπορείτε να κάνετε γυμναστική στην περιοχή σας. Ο/Η υπάλληλος του δήμου σάς λέει ότι υπάρχει ένα δημοτικό γυμναστήριο και ένα ωραίο στάδιο στο δημοτικό πάρκο. Εσείς λέτε ότι προτιμάτε να πάτε στο στάδιο, για να τρέχετε μέσα σε φυσικό περιβάλλον και να παίρνετε αέρα. Ο/Η υπάλληλος, όμως, προσπαθεί να σας πείσει ότι θα είναι καλύτερα στο γυμναστήριο και σας εξηγεί γιατί. Εσείς επιμένετε στην αρχική σας επιλογή.

**Β Ρόλος:**
Εργάζεστε ως υπάλληλος στην υπηρεσία αθλητισμού του δήμου και ένας κύριος/μια κυρία σας τηλεφωνεί, για να πάρει πληροφορίες σχετικά με χώρους όπου μπορεί να κάνει γυμναστική. Του/Της λέτε ότι υπάρχει ένα δημοτικό γυμναστήριο και ένα ωραίο στάδιο στο δημοτικό πάρκο. Εκείνος/Εκείνη σας λέει ότι προτιμάει να πάει στο στάδιο και σας εξηγεί γιατί. Εσείς προσπαθείτε να τον/την πείσετε ότι θα είναι καλύτερα στο γυμναστήριο, επειδή θα μπει σε ένα πρόγραμμα γυμναστικής που θα τον/την βοηθήσει πολύ. Εκείνος/Εκείνη, όμως, επιμένει στην αρχική του/της επιλογή.

### 5. Αγορά κινητού τηλεφώνου

**Α Ρόλος:**
Χρειάζεστε ένα καινούργιο κινητό τηλέφωνο. Εσείς, θέλετε να αγοράσετε ένα οικονομικό τηλέφωνο, αλλά με το οποίο να μπορείτε να παίζετε και παιχνίδια. Στο κατάστημα σάς λένε ότι υπάρχει ένα οικονομικό, το οποίο δεν περιέχει παιχνίδια, αλλά με το οποίο μπορείτε πιο εύκολα να συνδεθείτε στο διαδίκτυο και να προσθέσετε πιο πολλά ονόματα στις επαφές σας. Υπάρχει, όμως, και ένα πιο ακριβό, το οποίο έχει μεγάλη επιλογή από παιχνίδια. Εσείς θέλετε να πάρετε το δεύτερο, αλλά ο αδερφός/η αδερφή σας που είναι μαζί σας σας λέει να πάρετε το πρώτο και σας εξηγεί γιατί. Εσείς επιμένετε στην αρχική σας επιλογή.

**Β Ρόλος:**
Ο μικρός αδελφός/Η μικρή αδερφή σας χρειάζεται ένα καινούργιο κινητό τηλέφωνο. Στο κατάστημα σάς λένε ότι υπάρχει ένα οικονομικό, το οποίο, όμως, δεν περιέχει παιχνίδια και ένα πιο ακριβό, με το οποίο μπορεί να παίξει κανείς πολλά παιχνίδια. Ο αδερφός/Η αδερφή σας που είναι μαζί σας θέλει να αγοράσει το δεύτερο, αλλά εσείς προσπαθείτε να τον/την πείσετε να αγοράσετε το πρώτο και του/της εξηγείτε γιατί. Εκείνος/Εκείνη επιμένει στην αρχική του/της επιλογή.

### 6. Καινούργιος υπολογιστής ή επιδιόρθωση του παλιού;

**Α Ρόλος:**
Συζητάτε με έναν/μια υπάλληλο καταστήματος ηλεκτρονικών υπολογιστών, αν χρειάζεται να αγοράσετε καινούριο υπολογιστή ή, αν είναι καλύτερα να προσθέσετε καινούρια προγράμματα στον υπολογιστή σας. Εσείς πιστεύετε ότι, αν μεγαλώσετε τη μνήμη του υπολογιστή που έχετε, θα λειτουργεί πολύ καλά. Ο/Η υπάλληλος, όμως, προσπαθεί να σας πείσει ότι σας συμφέρει να πάρετε έναν καινούριο υπολογιστή με πιο σύγχρονα προγράμματα. Εσείς θεωρείτε ότι αυτό θα σας κοστίσει περισσότερα χρήματα και επαναλαμβάνετε ότι θέλετε να βελτιώσετε τις δυνατότητες τον υπολογιστή που έχετε με οικονομικό τρόπο.

**Β Ρόλος:**
Εργάζεστε ως υπάλληλος σε ένα κατάστημα ηλεκτρονικών υπολογιστών και ένας πελάτης/μια πελάτισσα σάς ρωτάει αν χρειάζεται να αγοράσει καινούριο υπολογιστή ή, αν είναι καλύτερα να προσθέσει καινούρια προγράμματα στον υπολογιστή που έχει. Εκείνος/Εκείνη πιστεύει ότι, αν μεγαλώσει τη μνήμη του υπολογιστή που έχει, θα λειτουργεί πολύ καλά. Εσείς, όμως, προσπαθείτε να τον/την πείσετε ότι τον/την συμφέρει να πάρει έναν καινούριο υπολογιστή με πιο σύγχρονα προγράμματα. Εκείνος/Εκείνη θεωρεί ότι αυτό θα του/της κοστίσει περισσότερα χρήματα και επαναλαμβάνει ότι θέλει να αναβαθμίσει τον υπολογιστή που έχει με οικονομικό τρόπο. Προσπαθείτε να του/της αλλάξετε τη γνώμη.

### 7. Εκμάθηση ξένης γλώσσας

**Α Ρόλος:**
Έχετε ψάξει στο διαδίκτυο και έχετε βρει πληροφορίες σχετικά με το πώς μπορείτε να μάθετε μια ξένη γλώσσα μέσα από ένα διαδικτυακό πρόγραμμα. Συζητάτε με έναν φίλο/μια φίλη σας για την ιδέα σας και τον/τη ρωτάτε να σας συμβουλεύσει. Εσείς παρουσιάζετε τα προτερήματα ενός τέτοιου τρόπου εκμάθησης.

**Β Ρόλος:**
Ο φίλος/Η φίλη σας θέλει να μάθει μια ξένη γλώσσα μέσω ενός διαδικτυακού προγράμματος. Εσείς πιστεύετε ότι ένας τέτοιος τρόπος εκμάθησης είναι πολύ δύσκολος και απαιτεί πολύ χρόνο. Επίσης, θεωρείτε σημαντική την προσωπική επαφή με έναν εκπαιδευτικό. Παρουσιάζετε, λοιπόν, τα προβλήματα ενός τέτοιου τρόπου εκμάθησης και προσπαθείτε να αλλάξετε τη γνώμη του φίλου/της φίλης σας.
